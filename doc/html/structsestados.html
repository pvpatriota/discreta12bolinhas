Detailed Description

  • Estruturas utilizadas para o correto funcionamento do programa.
  • A estrutura sestados cujo o nome definido foi estados, irá armazenar as informacoes sobre os estados da rede de petri.
    Parameters

    [in]-Nt recebe o numero de tokens de cada estado.
    [out]-O ponteiro prox aponta para o elemento seguinte da lista. Este ponteiro também é uma variável da estrutura sestados.
